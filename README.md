## Getting Started
To get started with the DOF Synthesis project, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/DOF-Synthesis.git`
2. Install dependencies: `npm install` or `yarn install`
3. Start the server: `npm start` or `yarn start`
4. Interact with the live demo: use the `curl` commands above or visit the server URL in your browser

## Overview
The Deterministic Observability Framework (DOF) is an autonomous AI agent that runs 24/7, utilizing A2A v0.3.0, MCP 2025-06-18, x402, and ERC-8004 protocols. The agent is designed to provide a secure and transparent framework for observability, with a focus on deterministic behavior and immutable proof storage on the Avalanche mainnet.

## What it does
The DOF agent is capable of performing the following tasks:

* Autonomous health checks every 30 minutes
* On-chain attestations using ERC-8004
* Immutable proof storage on the Avalanche mainnet
* Solidity security audits powered by Groq llama-3.3-70b

## Live Demo
You can interact with the DOF agent using the following `curl` commands:

* Health check: `curl -X GET https://dof-hackathon.io/health`
* Attestation: `curl -X POST https://dof-hackathon.io/attest`
* Proof retrieval: `curl -X GET https://dof-hackathon.io/proof`

## Architecture
The DOF architecture consists of the following components:

* Autonomous AI agent (Agent #1686 on ERC-8004 registry)
* Solidity security audits (Groq llama-3.3-70b)
* On-chain proof storage (Avalanche mainnet)
* LLM providers (Groq, Cerebras, NVIDIA, OpenRouter, SambaNova, MiniMax)

## On-Chain Evidence
The DOF agent has performed over 40 on-chain attestations on the Avalanche mainnet, with a 0% false positive rate (FPR) across 12,229 Garak adversarial payloads.

## Quick Start
To get started with the DOF agent, follow these steps:

1. Clone the repository: `git clone https://github.com/DOF-Hackathon/DOF.git`
2. Install dependencies: `npm install`
3. Start the agent: `npm start`
4. Interact with the agent using the live demo `curl` commands

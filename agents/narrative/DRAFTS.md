# Drafts — Content Pipeline

## Tipos de Contenido
| Tipo | Frecuencia | Canal |
|------|-----------|-------|
| Thread Twitter/X | 3x semana | X |
| Blog update | 1x semana | Blog/Mirror |
| Grant narrative | Por demanda | Grants |
| Newsletter | 1x semana | Email |
| Pitch deck | Por demanda | Investors |

## Calendario Semanal
- **Lunes**: Thread de progreso semanal
- **Miercoles**: Thread educativo/tendencias
- **Viernes**: Thread de comunidad/engagement
- **Domingo**: Blog post / newsletter

## Template Thread (6-12 tweets)
1. Hook viral (pregunta o dato impactante)
2. Contexto del problema
3. Nuestra solucion (1-2 tweets)
4. Datos/metricas de traccion
5. Que sigue (roadmap)
6. CTA (follow, join, try)

## Output
Archivo: output/{project}/content_YYYYMMDD_HHMMSS.md
